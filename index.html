Modul Materi Belajar

Pilih topik di bawah ini untuk memulai membaca rangkuman teori pemrograman web yang dikemas secara padat dan jelas.

1. Pengenalan Dokumen HTML

HTML (HyperText Markup Language) adalah bahasa markup standar untuk membuat halaman web. HTML mendefinisikan struktur halaman web Anda.

<!DOCTYPE html>
<html>
<head>
  <title>Halaman Saya</title>
</head>
<body>
  <h1>Halo Dunia</h1>
</body>
</html>
Mudah

2. Elemen, Tag, & Atribut

Elemen HTML ditulis dengan tag pembuka, konten, dan tag penutup. Atribut memberikan informasi tambahan tentang elemen tersebut (ditulis di tag pembuka).

<!-- Tag dengan Atribut href -->
<a href="https://google.com">Buka Google</a>

<!-- Tag Gambar dengan src dan alt -->
<img src="gambar.jpg" alt="Pemandangan">
Mudah

3. Form & Input

Form digunakan untuk mengumpulkan input dari user. Terdiri dari berbagai elemen seperti text input, password, checkbox, radio, dan tombol submit.

<form action="/submit">
  <label>Nama: </label>
  <input type="text" name="nama">
  <button type="submit">Kirim</button>
</form>
Sedang

1. Pengenalan CSS & Selector

CSS (Cascading Style Sheets) digunakan untuk mengatur gaya visual halaman web (warna, font, tata letak). Selector menentukan elemen mana yang ingin diberi gaya.

/* Menargetkan semua tag h1 */
h1 {
  color: #ff6b00;
  font-size: 24px;
}

/* Menargetkan class (.nama-class) */
.box-warna {
  background-color: #151821;
}
Mudah

2. CSS Box Model

Setiap elemen HTML dianggap sebagai sebuah kotak ("box"). Box Model terdiri dari: Content (isi), Padding (jarak dalam), Border (garis tepi), dan Margin (jarak luar).

.card-box {
  width: 300px;
  padding: 20px;
  border: 2px solid #ff6b00;
  margin: 15px;
}
Mudah

3. Flexbox Layout

CSS Flexbox mempermudah pengaturan tata letak satu dimensi secara dinamis dan responsif, baik baris (row) maupun kolom (column).

.flex-container {
  display: flex;
  justify-content: space-between;
  align-items: center;
  background-color: #151821;
  padding: 10px;
}
Sedang

1. Variabel & Tipe Data

JavaScript digunakan untuk logika interaktif. Variabel menyimpan data. Ada 3 cara deklarasi: var, let, dan const. Tipe datanya meliputi string, number, boolean, array, dll.

let nama = "Budi";
const umur = 20;
let sudahLulus = false;
let hobi = ["coding", "gaming"];
Mudah

2. DOM Manipulation

DOM (Document Object Model) memungkinkan JS mengakses dan mengubah elemen HTML & CSS secara dinamis saat web sedang berjalan.

// Mengambil elemen h1 berdasarkan ID
const judul = document.getElementById("title");

// Mengubah teks dan warna
judul.innerText = "Selamat Belajar!";
judul.style.color = "#ff6b00";
Sedang

3. Event Listener

Event listener digunakan agar web merespon tindakan user, seperti klik tombol, ketikan keyboard, submit formulir, atau scroll halaman.

const tombol = document.querySelector("#myBtn");

tombol.addEventListener("click", () => {
  alert("Tombol berhasil diklik!");
});
Sedang
